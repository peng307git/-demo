    Innovative blockchain compression technology can maximally compress the capacity of blockchain database and save the valuable blockchain space.     At the operational level, the innovative search engine technology called Script2TxPosDB, can search the balance of each instantaneous address, and realize instantaneous switching between multiple bank accounts, without the need to restart the client or refresh the account balance by scanning the blockchain for a long time.

    2. Main functions and applications of Mind Bank

    2.1 System trust interest

    The Mind Bank system is set a function of Lock Time namely “Lock Time into Chain”, the mind gene locked into locktime can obtain a corresponding return which is the system trust interest. System users make the actions of storing the mind gene, and such actions are recorded in the public blockchain and a global public contract is formed. The system trust interest is a type of rewards given to users based on their trusts on the Mind Asset values.

    The trust period of Mind Bank system trust interest ranges from 16 ~1024 days. The longer the locktime is, the higher the interest returns will be. The highest interest rate per hundred days in the first stage is9.99999%. The remaining 10% of the total amount under the current stage will automatically enter into the next stage, and the corresponding interest rate of the each storage cycle will be decreased by 10%.

    Unlike the interest payment form of “regular deposit” in real banks, the system trust interests of Mind Bank is paid by the way “Pre-”. That is to say, when system users lock MG into locktime, the system interest will automatically be deposited into individual accounts, and MG and its interests gained can only be withdrawn and used after unlocking the trust block.

    2.2 Blockchain browser

    The blockchain browser is a block query tool of M.A Blockchain embedded in the Mind Bank, which can realize the real-time updates of M.A Blockchain data through the integration of block data in three dimensions of block height (i.e. number of blocks), bank address and each transaction number, and is convenient for system users to instant check the transaction information of blockchain and also guarantees all data in the decentralized system open and transparent.

    2.3 Commonweal voting and donation mechanism

    There is a commonweal vote mechanism in Mind Bank. Users can get corresponding number of votes based on MG “coin age” under individual bank’s address, which allows them to conduct global commonweal vote in each century. At the end of the current century, the system will select the commonweal organization which gets the most votes in this century. One commonweal organization will be selected each century. Fifty percent of global network transaction fees in next century and commonweal donations from global system users will be distributed to the selected commonweal organizations equally.

    The Mind Bank has set a commonweal donation mechanism with one key. System users can voluntarily choose the amount of donated MG which will be distributed equally to the commonweal organizations selected by the global commonweal votes in the current century.
